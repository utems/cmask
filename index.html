CMask

CMask source, pre-compiled binary distributions, and install scripts

Mac Installation - Automagic

To install cmask on your mac system, just hit up our automagical script. Open Terminal (found in /Applications/Utilities) and copy paste the following command:

curl http://utems.github.com/cmask/cmask-install.sh | sh

This will figure out if your machine is capable of compiling from source or if it should just download the pre-compiled binaries. Then it will install them in /usr/local/bin and add it to your path. Voila!
